# Month 1: C Fundamentals (Days 1-30)

## Overview
Foundation of C programming covering syntax, control flow, arrays, and basic file operations.

## Week 1: Getting Started
- **Day 1-3**: Environment setup, first C program, compilation basics
- **Day 4-7**: Variables, data types, operators, input/output

## Week 2: Control Flow & Arrays
- **Day 8-10**: if/else, switch statements
- **Day 11-14**: Loops (for, while, do-while), break/continue
- **Day 15-17**: Arrays, multi-dimensional arrays

## Week 3: Functions & File I/O
- **Day 18-21**: Functions, parameters, return values, scope
- **Day 22-24**: Strings, string manipulation
- **Day 25-27**: File I/O (fopen, fclose, fread, fwrite)
- **Day 28-30**: Structures basics, enums, typedef

## Week 4: Practice & Integration
- Applied concepts through practical exercises
- String manipulation and array operations
- Basic pointer introduction
- ML-related calculations (vectors, matrices)

## Key Concepts Mastered
- C syntax and compilation
- Control structures
- Arrays and strings
- Functions
- File operations
- Basic structures
